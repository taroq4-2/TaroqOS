# 🛡️ Taroq OS — حاجب إعلانات ذكي لأندرويد

[![Build APK](https://github.com/taroq4-2/TaroqOS/actions/workflows/build.yml/badge.svg)](https://github.com/taroq4-2/TaroqOS/actions/workflows/build.yml)
[![License: Apache 2.0](https://img.shields.io/badge/License-Apache_2.0-blue.svg)](LICENSE)

**Taroq OS** هو تطبيق أندرويد مفتوح المصدر يحجب الإعلانات والتتبع بدون روت، يعمل عبر VPN محلي.

---

## ✨ المميزات

- 🚫 **حجب 250+ شبكة إعلانية** (Google Ads, Meta, TikTok, Snap, Amazon...)
- 🔒 **بدون Root** — يعمل على أي جهاز أندرويد 8+
- 🌐 **VPN محلي** — لا يوجه أي بيانات خارج الجهاز
- ⚡ **صفر بطاريت** — تصفية DNS خفيفة لا تستهلك الموارد
- 📊 **إحصائيات حية** — كم إعلان تم حجبه
- 🔄 **تشغيل تلقائي** عند إقلاع الجهاز

---

## 🧠 التقنية المبتكرة

بدلاً من القوائم التقليدية البسيطة، يعتمد Taroq OS على:

1. **تحليل أنماط النطاقات** — يكشف شبكات الإعلانات الجديدة تلقائياً
2. **فلترة DNS ذكية** — يعترض استعلامات DNS فقط، لا يقطع الإنترنت
3. **مطابقة Regex** — يحجب النطاقات ذات الأنماط الإعلانية حتى لو لم تكن مدرجة
